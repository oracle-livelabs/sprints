🔌 The shift MCP makes

AI tools were stranded outside enterprise data. Every agent needed its own custom connector — built from scratch, every time.
MCP opens the door — with Oracle already holding the key. One protocol. Any agent. Integrate once and you're done.
MCP = Model Context Protocol. Marketed as "USB-C for AI applications" — a singular interface that hides the complexity of the server implementation.

⚙️ No new server. SQLcl is the runtime.

• SQLcl 25.4 ships with an MCP Server built in
• Runs locally on STDIO — no cloud dependency
• Uses JSON-RPC 2.0 — standard, auditable protocol

Works with any Oracle Database where SQLcl is supported — including on-premises, Cloud, and containers. Editions from FREE upward. Always check compatibility for your specific version.

Fewer queries to "figure stuff out." Faster responses. Fewer tokens consumed — which directly reduces AI inference cost.
